## Installation


### Manuel Installation

1. Clone the repo
   ```sh
    git clone https://github.com/AnaFernandezAsix/CyberShield2
   ```
   ```sh
    cd CyberShield2
   ```
2. Build docker image
   ```sh
    sudo docker build -t ana/cyber2 .
   ```
3. Run container
   ```sh
    sudo docker run -d -p 1337:80 cyber2
   ```
4. Go to http://localhost:1337
